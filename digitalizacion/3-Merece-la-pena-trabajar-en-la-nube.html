3. ¿Merece la pena trabajar en la nube?

3.1 ¿Cuánto cuesta implantar el cloud computing?

En el sistema tradicional, las empresas necesitaban comprar servidores físicos mediante un proceso de aprovisionamiento que podía durar meses. Escalar este proceso cuando había picos de demanda, o si el negocio crecía rápidamente, era muy difícil. El cloud computing solucionó estos problemas ofreciendo recursos de computación escalables bajo demanda.

3.2 Ventajas del cloud computing

Las empresas pueden contratar más servidores a medida que los necesitan, disponiendo de tanta potencia de proceso de datos como requieran.

Las empresas alojan en la nube copias de seguridad de sus recursos digitales.

Las empresas pueden acceder a dus datos de forma segura y desde cualquier dispositivo, en cualquier ubicación geográfica.

La nube ofrece herramientas y plataformas para crear y probar aplicaciones, lo que reduce el tiempo de lanzamiento.

El cloud computing cuenta con recursos casi ilimitados para procesar grandes volúmenes de datos con el fin de agilizar los procesos de investigación y reducir el tiempo necesario para obtener información valiosa.

Los servicios de cloud computing ofrecen medidas de protección frente a los ciberataques y ante la posible pérdida/borrado de datos sensibles.
